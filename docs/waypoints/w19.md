# W19: Fix Discover Skill Invocation Bug

## Objective

Fix the error where invoking `do-stuff-helper:discover` via the Skill tool returns a `disable-model-invocation` error.

## Done When

- Root cause confirmed (name collision between command and skill)
- The discover command is removed
- The discover skill can be invoked via the Skill tool without error
- References to `/discover` updated in CLAUDE.md and other skills

## Design

Delete `commands/discover.md`. The command only wraps the skill and introduces a name collision that causes the `disable-model-invocation` error. Update any references to `/discover` in CLAUDE.md and skill docs to use `do-stuff-helper:discover` instead.
