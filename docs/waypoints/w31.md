# W31: Front-End Design Plugin in Organize

## Objective

Ensure every new activity bootstrapped by the organize skill has the `frontend-design` plugin installed, so agents working on UI tasks have design system guidance available from the start.

## Done When

- The organize skill's Step 2 includes `frontend-design` in the plugin install list
- New activities organized after this change have `frontend-design` installed at project level
- No CLAUDE.md template changes needed — installed plugins are auto-discovered by new sessions

## Implementation Notes

Add one entry to `skills/organize/SKILL.md` Step 2's plugin list:

```
4. **frontend-design** — `claude plugin install frontend-design@claude-plugins-official`
   Provides design system guidance for activities that involve building user interfaces.
```

No conditional detection needed — installing it for all activities is cheap (it's just a skill file) and avoids the complexity of trying to predict which activities will need UI work.
